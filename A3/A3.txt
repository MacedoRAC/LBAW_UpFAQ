====== Protótipo da Interface com o Utilizador ======

Este artefacto, o Protótipo da Interface com o Utilizador (ou protótipo horizontal) tem dois objetivos:
  * ajudar a identificar e descrever os requisitos do utilizador;
  * dar uma ideia de como será a interface com o utilizador (IU) do produto a desenvolver e suscitar novos requisitos.

===== - Interface e princípios gerais =====


{{:lbaw1444:imagem_do_website1.png?200|}}\\
Figura 1: Imagem do website

Elementos Comuns a todas as páginas:
  * optou-se pela cor azul e branco para construir todo o website
  *  o website adapta-se a todos os tipos de tamanho(PC's, telemóveis e tablets)
  * os pontos comuns às várias páginas (por exemplo a ligação à página principal, o campo de procura ) permanecem na mesma posição relativa
  * utilizamos vários eventos de "touch" para facilitar a interação com o website nos dispositivos móveis(por exemplo para mostrar/ocultar a lista de utilizadores)

O //Sitemap// é uma representação hierárquica da estrutura de um site, composta por páginas web.
Desta forma, é fornecida à equipa de projeto uma visão de como o sítio vai ser construído, ajudando nomeadamente a esclarecer a hierarquia de informação [[biblio|[STMAP] ]].

===== - Interações =====

São descritas as sequências de passos das principais interações com o sistema. O intuito é fornecer uma visão clara da utilização do sistema. 

São apresentados fluxogramas para uma maior compreensão dos processos de autenticação e de navegação na página de perfil. De notar que não são revelados detalhes da interação, mas apenas uma visão geral da utilização e comportamento do sistema.
